02. Tag Mapping

Best Mapper?

  • Exact: All aligners performed well. BWA mem performs noticeably worst.
  • Snp: All aligners comparable if mismatch rate<4% except for SOAP.
  • Insertion: BWA mem best. With increasing read length, Bowtie2 catches up to STAR.
  • Deletion: Similar observations to insertion.
  • Summary: Overall, Bwa mem and Bowtie2. STAR has higher error rate.
